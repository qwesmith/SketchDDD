# sketchddd-viz

Visualization generation for [SketchDDD](https://github.com/ibrahimcesar/SketchDDD) domain models.

## Supported Formats

- **Graphviz DOT**: For rendering with Graphviz
- **Mermaid**: For rendering in Markdown/GitHub
