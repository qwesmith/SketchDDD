# sketchddd-codegen

Code generation for [SketchDDD](https://github.com/ibrahimcesar/SketchDDD) domain models.

## Supported Targets

- **Rust**: Structs, enums, and validation
- **TypeScript**: Interfaces, types, and Zod schemas
- **Kotlin**: Data classes and sealed classes
